# Bratři v triku

Statický projekt nad rešerší studia Bratři v triku. Web je záměrně bez frameworku a build kroku, aby šel přímo nasadit na Vercel nebo GitHub Pages.

## Obsah

- `index.html` – veřejná prezentační stránka
- `styles.css` – stylování webu
- `Bratri_v_triku_research.md` – plná projektová rešerše
- `vercel.json` – základní konfigurace pro Vercel

## Lokální kontrola

```bash
npm run check
```

## Deployment

Projekt je připravený jako čistě statický web. Na Vercelu není potřeba build command ani output directory.
